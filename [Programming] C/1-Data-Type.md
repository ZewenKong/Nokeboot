# Data type

| Data type | CN name             | printf                  |
| --------- | ------------------- | ----------------------- |
| char      | 字符                | `%c` 打印字符           |
| int       | 整型                | `%d` 打印整型十进制数据 |
| short     | 短整型              |                         |
| long      | 长整型              |                         |
| long long | 长长整型 (弹钢琴的) |                         |
| float     | 单精度浮点数        | `%f` 打印浮点数         |
| double    | 双精度浮点数        | `%lf ` 打印双精度浮点数 |
|           |                     | `%p` 以地址的形式打印   |
|           |                     | `%x` 打印十六进制       |
